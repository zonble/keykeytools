使用說明

這個工具可以幫助您將微軟新注音輸入法中所建立的自訂詞庫，轉換成 Yahoo! 奇摩輸入法中的「好打注音」輸入法詞庫。請按照以下步驟操作－

從新注音輸入法中匯出詞庫

微軟新注音輸入法提供兩種使用者自行加入新詞的方式－一是在輸入時、輸入緩區中還有內容的狀況下，按住鍵盤上的 Shift 按鍵，然後按下左右鍵移動輸入游標，這時候，輸入區下方會出現一條較粗的底線，這條底線的範圍，就是要加入新詞的範圍；在按下了 Enter 鍵之後，便可以加入新詞。另外一種方式，則是叫出新注音輸入法的「使用者造詞工具」。

要將新注音輸入法的自訂詞庫轉換到 Yahoo! 奇摩輸入法的好打注音使用，第一步自然是要先匯出新注音輸入法的詞庫。首先請從新注音輸入法的工具列中，選擇「工具選單」，選擇「使用者造詞」，便可以叫出「使用者造詞工具」。

在「使用者造詞工具」中，請選擇「檔案」選單中的「匯出」，便可以透過純文字檔案的方式，匯出所有已經建立的使用者自訂詞庫。

請注意：因為新注音輸入法的版本相當多，在寫這個工具的時候，無法找到所有版本的新注音輸入法測試，這個工具只測試過 2002、2003（Windows XP 平台），以及 2007 版（Windows Vista 內建，或購買 Office 2007 附贈）。

上傳詞庫並轉換

請將剛剛匯出的文字檔，透過本頁上方的表格上傳轉換。上傳成功之後，會自動產生一個供 Yahoo! 奇摩輸入法使用的純文字檔案。

將轉出的詞庫匯入 Yahoo! 奇摩輸入法

轉出的純文字檔案通用於 Yahoo! 奇摩輸入法的 Windows 版本與 Mac OS X 版本－

Windows 版本

請從 Yahoo! 奇摩輸入法的工具列中，點選最右邊的工具（搥子）圖示，從工具選單中選擇「偏好設定」，從「偏好設定」中選擇「詞彙」，便可以看到「匯入自訂詞詞庫」按鈕。

按下「匯入自訂詞詞庫」按鈕後，便會詢問您想要匯入哪個檔案，請選擇剛剛下載的檔案，便可完成匯入。

如果您想要確認是否正確匯入，您也可以在同一頁當中，按下「啟動詞彙編輯工具」按鈕，叫出 Yahoo! 奇摩輸入法的詞彙編輯工具，在這個工具的編輯畫面中，檢查是否出現剛才匯入的詞彙。

Mac OS X 版本

Mac OS X 版本的匯入方式，可說與 Windows 版本完全相同。

當您在使用 Yahoo! 奇摩輸入法時，按下畫面右上方選單列（menu bar）上的 Yahoo! 奇摩輸入法圖示，便可以叫出輸入法選單，從輸入法選單中，便可以叫出偏好設定工具。同樣的，您也可以在 Mac OS X 版本的偏好設定工具中，找到「匯入自訂詞詞庫」按鈕，匯入詞庫。
